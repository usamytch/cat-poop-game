## 📋 Обязательный порядок действий для каждого промпта

### Шаг 0 — Чтение README.md (ВСЕГДА ПЕРВЫЙ ШАГ)

```
read_file("README.md")
```

Агент обязан прочитать [`README.md`](../README.md) и убедиться, что понимает:

- **Геймплей**: механики кота, хозяина, шкалы срочности, лотка
- **Управление**: клавиши WASD, пробел, M, Enter, 1/2/3
- **Фичи**: паника-режим, оверлеи, бонусы, комбо-система, сложности
- **Технологии**: Vanilla JS, Canvas 2D, Web Audio API, localStorage, LCG RNG
- **Структуру файлов**: роль каждого модуля в `js/`

Только после прочтения README.md агент переходит к следующим шагам.

## 🔑 Ключевые правила из README.md, которые агент должен помнить

### Игровые механики
- Шкала срочности растёт каждый кадр: `urgeRate/60 * (1 + (level-1)*0.08)`
- Паника при `urge/maxUrge > 0.75`
- Авария при `urge >= maxUrge`
- Комбо при 3 попаданиях подряд → `owner.flee()`
- `hitUrgeReduce` всегда меньше прироста за кулдаун (нельзя читерить стрельбой)

### Сложности
| Режим | Ключ | Хозяин появляется |
|---|---|---|
| 😸 Лёгкий | `easy` | Со 2 уровня |
| 😼 Нормал | `normal` | С 1 уровня |
| 😈 Хаос | `chaos` | С 1 уровня |

### Бонусы
| Бонус | Ключ | Эффект |
|---|---|---|
| 🐟 Рыбка | `fish` | Ускорение x1.7, 5 сек |
| 🧶 Клубок | `yarn` | Хозяин замирает, 5 сек |
| 💊 Таблетка | `pill` | -30% срочности мгновенно |

### Технические ограничения
- Только Vanilla JS — никаких фреймворков и npm-пакетов
- Весь звук — процедурный через Web Audio API
- Вся графика — Canvas 2D
- Прогресс — только через `localStorage`
- Запуск: открыть [`index.html`](../index.html) в браузере (или `python3 -m http.server 8080`)

# 🐱 Cat Poop Game — Бэнгер Edition

> Доведи кота до лотка — пока не поздно!

Браузерная аркада на чистом JavaScript + Canvas 2D. Кот хочет в туалет. Хозяин не даёт. Шкала срочности заполняется. Успей добраться до лотка.

**[▶ Играть](index.html)** — открой `index.html` в браузере, никакой установки не нужно.

---

## 🕹️ Управление

| Клавиша | Действие |
|---|---|
| `WASD` / `←↑↓→` | Движение кота |
| `Пробел` / `X` | Выстрел какашкой |
| `1` / `2` / `3` | Обучение / Нормал / Хаос |
| `Enter` | Старт / Меню |
| `M` | Вкл/выкл звук |
| `Escape` / `P` | Пауза / продолжить |

---

## 🎮 Как играть

- Уворачивайся от хозяина и препятствий
- Шкала **ХОЧЕТСЯ** (срочность) постоянно растёт — при 100% случается **авария 💩**
- Стреляй какашками в хозяина — 3 попадания подряд = **COMBO!** и хозяин убегает
- Собирай бонусы: 🐟 ускорение, 🧶 заморозка хозяина, 💊 снижение срочности, ❤️ жизнь, 🌿 котовник
- Доберись до **лотка 🐾** — переходи на следующий уровень

---

## 🎓 Режимы

| Режим | Описание |
|---|---|
| 🎓 Обучение | 3 фиксированных экрана: движение и лоток → хозяин и COMBO → экзамен с бонусами и panic. Без рекордов и потери статистики |
| 😼 Нормал | Основная честная игра; хозяин появляется со 2 уровня |
| 😈 Хаос | Быстрая срочность и агрессивный хозяин с 1 уровня |

Обучение всегда можно повторить. После первого прохождения стартовый выбор по умолчанию переключается на Normal.

## 🗺️ Локации

5 обычных локаций (🛋️ Зал, 🚿 Ванная, 🍳 Кухня, 🌳 Двор, 🏡 Дача) идут **5-уровневыми актами**: внутри акта сложность нарастает до пика на 5/5, затем новая локация даёт короткий выдох. Повторные круги получают вариант локации вроде `Зал II`.

Секретная **🕸️ Подвал**-аномалия с лабиринтом по-прежнему появляется с уровня 9 и может хаотично перебить обычный акт.

Процедурные карты проходят проверку честности: генератор учитывает длину и временной бюджет пути, зоны движения препятствий, безопасность спавна и лотка, а бонусы размещает только в достижимых боковых маршрутах. Геометрия зависит от seed забега и номера уровня, но не от счёта; seed показывается в финале, а `R` повторяет тот же забег.

У каждой локации своя оригинальная процедурная тема: домашний фолк в Зале, морской вальс в Ванной, кухонный свинг, мультяшная погоня во Дворе, приключенческий марш на Даче и персидский стелс в Подвале. При панике текущая тема ускоряется и проигрывается в обратном порядке.

## 🚀 Запуск

```bash
# Просто открой в браузере:
open index.html

# Или через локальный сервер (рекомендуется для звука):
python3 -m http.server 8080
# затем открой http://localhost:8080
```

> Web Audio API требует пользовательского взаимодействия. Звук заработает после первого нажатия клавиши.

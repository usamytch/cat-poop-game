# Cat Poop Game

Весёлая браузерная arcade-игра про кота, который очень хочет добраться до лотка, пока его не поймал хозяин.

## Об игре

[`Cat Poop Game`](README.md) — это небольшая canvas-игра с комедийной идеей, простым управлением и постепенно растущей сложностью. Игрок управляет котом, избегает хозяина, обходит препятствия, следит за шкалой срочности и пытается вовремя добраться до лотка.

В игре уже есть:
- несколько локаций;
- процедурная генерация препятствий;
- рост сложности по уровням;
- преследующий хозяин;
- возможность отбиваться меткими 💩;
- экран победы и поражения;
- HUD с основной игровой информацией.

## Геймплей

Цель каждого уровня — успеть добраться до лотка до того, как шкала желания заполнится полностью.

По пути игроку мешают:
- хозяин, который пытается поймать кота;
- мебель и другие препятствия;
- растущая скорость и напряжение на следующих уровнях.

После успешного похода в лоток игрок получает очки, переходит на следующий уровень и попадает в новую локацию.

## Управление

- `WASD` / стрелки — движение
- `Пробел` — бросить 💩
- `Enter` — перейти дальше после победы или начать заново после поражения

Также поддерживаются русские клавиши для движения.

## Запуск

Проект не требует сборки.

### Вариант 1
Открой [`index.html`](index.html) в браузере.

### Вариант 2
Запусти любой простой локальный сервер, например:

```bash
python3 -m http.server 8000
```

После этого открой `http://localhost:8000`.

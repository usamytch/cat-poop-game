# 🐱 Cat Poop Game — Бэнгер Edition

> Доведи кота до лотка — пока не поздно!

Браузерная аркада на чистом JavaScript + Canvas 2D. Кот хочет в туалет. Хозяин не даёт. Шкала срочности заполняется. Успей добраться до лотка.

---

## 🎮 Геймплей

- Управляй котом по комнате, уворачиваясь от хозяина и препятствий
- Шкала **срочности** постоянно растёт — при 100% случается **авария 💩**
- Хозяин преследует кота — при поимке игра заканчивается
- Стреляй какашками в хозяина, чтобы его остановить
- Собирай бонусы на карте
- Доберись до **лотка 🐾** — и переходи на следующий уровень

---

## 🕹️ Управление

| Клавиша | Действие |
|---|---|
| `WASD` / `←↑↓→` | Движение кота |
| `Пробел` / `X` | Выстрел какашкой в направлении движения |
| `1` / `2` / `3` | Выбор сложности на стартовом экране |
| `Enter` | Старт / Возврат в меню |
| `M` | Вкл/выкл звук |

## ✨ Фичи

### 🔊 Звуки (Web Audio API)
Все звуки генерируются процедурно через осцилляторы — никаких внешних файлов:
- Мяу при старте, пуканье при выстреле, попадание, тревога, победа, провал, комбо, подбор бонуса

### 😱 Паника-режим
При срочности > 75%:
- Шкала мигает красным с пульсирующей рамкой
- Лоток светится красным
- Экран трясётся
- Тревожный звук каждые полсекунды

### 🎉 Оверлеи
- **Победа** — конфетти из 💩⭐✨🎉 с физикой
- **Авария** — лужа 💧 с брызгами
- **Поимка** — тёмный оверлей с анимацией

### 🎁 Бонусы
| Бонус | Эффект | Длительность |
|---|---|---|
| 🐟 Рыбка | Ускорение кота x1.7 | 5 сек |
| 🧶 Клубок | Хозяин замирает | 5 сек |
| 💊 Таблетка | -30% срочности | мгновенно |

### 💥 Комбо-система и какашки на лице
3 попадания в хозяина подряд = **COMBO!** + автоматическая заморозка хозяина + звук

Каждое попадание оставляет 💩 на лице хозяина — они накапливаются и видны во время бегства.
После того как хозяин вернулся из угла (флаг бегства истёк), все какашки с лица исчезают.

### 🎯 Снижение срочности при стрельбе
Каждое попадание в хозяина немного снижает шкалу срочности кота.
Величина снижения зависит от сложности и настраивается в `DIFF[difficulty].hitUrgeReduce`:

| Режим | Снижение за попадание |
|---|---|
| 😸 Лёгкий | 1.5 |
| 😼 Нормал | 1.0 |
| � Хаос | 0.6 |

Снижение всегда меньше прироста срочности за кулдаун выстрела — читерить одной стрельбой нельзя.

### 😈 Режимы сложности
| Режим | Срочность | Хозяин | Появляется |
|---|---|---|---|
| 😸 Лёгкий | Медленная | Медленный | Со 2 уровня |
| 😼 Нормал | Стандарт | Средний | С 1 уровня |
| 😈 Хаос | Быстрая | Быстрый | С 1 уровня |

### 🏆 Прогресс
- Рекорд и статистика сохраняются в `localStorage`
- Отображаются на стартовом экране: рекорд, лучший уровень, поймано хозяином, аварий, какашек выпущено

### 🗺️ Локации и препятствия
5 уникальных локаций с палитрами и декорациями:
- **Зал** — шкафы, кресла, фикусы
- **Ванная** — раковины, унитазы, корзины
- **Кухня** — столы, холодильники, табуреты
- **Двор** — деревья, лавки, кусты
- **Загородный дом** — поленницы, бочки, кресла-качалки

С 5 уровня препятствия начинают **двигаться**.

### 🔢 Сеточная генерация уровней (Grid System)

Все препятствия, лоток и бонусы расставляются по **чёткой сетке 40×40 пикселей**:

```
Игровая зона: 1120 × 520 px → 28 колонок × 13 строк = 364 ячейки
```

- **Препятствия** — размер кратен 40px (2×2, 2×4, 4×2, 4×4 ячейки), позиция всегда выровнена по сетке
- **Нет наложений** — `occupiedCells` Set гарантирует, что ни одно препятствие не перекрывает другое
- **Лоток** — всегда на свободных ячейках, никогда не появляется на препятствии
- **Бонусы** — спавнятся в центрах свободных ячеек
- **Декор** — фоновые элементы (ковры, камни, заплатки) рисуются по сетке, но не блокируют движение
- **Детерминированность внутри сессии** — та же пара `level + score + globalSeed` всегда даёт одинаковую карту (LCG RNG)
- **Уникальность каждого запуска** — `globalSeed` берётся из `Date.now()` при старте игры, поэтому каждый новый запуск даёт уникальное расположение препятствий, лотка и бонусов

### 🤖 A* навигация хозяина

Хозяин использует **алгоритм A\*** по сетке для обхода препятствий:
- Путь пересчитывается каждые ~0.75 сек или при исчерпании текущего
- Хозяин плавно следует по waypoint'ам ячеек, скользя вдоль стен
- Больше никаких застреваний в углах — хозяин всегда находит путь к коту

## 🏗️ Технологии

- **Vanilla JS** — никаких фреймворков
- **Canvas 2D API** — вся графика
- **Web Audio API** — процедурный звук
- **requestAnimationFrame** — игровой цикл
- **localStorage** — сохранение прогресса
- **LCG Seeded RNG** — детерминированная генерация уровней
- **Vitest** — unit- и интеграционные тесты (Node.js, без браузера)

## 🚀 Запуск

Открой `index.html` в браузере — и играй. Никакой сборки, никаких зависимостей.

```bash
# Вариант через локальный сервер (рекомендуется для звука)
python3 -m http.server 8080
# затем открой http://localhost:8080
```

> **Примечание:** Web Audio API требует пользовательского взаимодействия для инициализации. Звук заработает после первого нажатия клавиши.
